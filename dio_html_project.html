Introdução

HTML, ou HyperText Markup Language, é a linguagem de marcação padrão usada para criar páginas da web. Foi desenvolvido por Tim Berners-Lee no final dos anos 1980 e início dos anos 1990.

História do HTML

Primeiros Anos

HTML foi criado como uma forma de compartilhar documentos científicos de maneira simples e eficaz. Em 1991, Berners-Lee publicou um documento chamado "HTML Tags" que descrevia 18 elementos que poderiam ser usados para formatar documentos. Este documento é considerado a primeira versão de HTML.

"The World Wide Web (WWW) project aims to allow all links to be made to any information anywhere." - Tim Berners-Lee

Expansão e Padrões

Nos anos seguintes, o HTML passou por várias revisões e evoluiu para incluir novos elementos e atributos. Em 1995, foi publicado o HTML 2.0, que incluiu melhorias significativas. (Fonte: Especificação HTML 2.0)

Utilização do HTML

Estrutura de um Documento HTML

Um documento HTML típico possui a seguinte estrutura básica:

  • DOCTYPE: Declara o tipo de documento.
  • Head: Contém metadados e informações como título e links para arquivos CSS.
  • Body: Contém o conteúdo visível da página.

Exemplo de Estrutura

<!DOCTYPE html>
<html lang="pt-BR">
<head>
<meta charset="UTF-8">
<title>Título da Página</title>
</head>
<body>
<h1>Olá, Mundo!</h1>
</body>
</html>

Elementos Comuns

Alguns elementos comuns usados em HTML incluem:

  1. Parágrafos: Criados com a tag <p>.
  2. Itálico: Criado com a tag <i>.
  3. Sublinhado: Criado com a tag <u>.
  4. Texto Deletado: Criado com a tag <del>.
  5. Texto Marcado: Criado com a tag <mark>.
  6. Texto Pequeno: Criado com a tag <small>.

Conclusão
